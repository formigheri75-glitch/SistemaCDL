    INSTITUIÇÃO

    -GET - Obter instituição por ID

    GET /instituicoes/<id>
    

    - POST - Criar instituição
    
    POST /instituicoes
    Content-Type: application/json

    {
        "usuarioID": 1,
        "nomeInstituicao": "Instituição X",

    USUÁRIO

    - GET - Listar todos os usuários
    
    GET /usuarios
    

    - GET - Obter usuário por ID
    
    GET /usuarios/<id>
    

    - POST - Criar usuário
    
    POST /usuarios
    Content-Type: application/json

    {
        "nome": "João Silva",
        "email": "joao@example.com",
        "senha": "senha123",
        "tipoUsuario": "Admin",
        "ativo": 1
    }
    

    - PUT - Atualizar usuário
    
    PUT /usuarios/<id>
    Content-Type: application/json

    {
        "nome": "João Silva Atualizado",
        "email": "joao.novo@example.com",
        "senha": "novaSenha123",
        "tipoUsuario": "User",
        "ativo": 1
    }
    

    - DELETE - Deletar usuário
    
    DELETE /usuarios/<id>
